# 🎱 Lottoria 3D Draw (로또리아 3D 추첨기)

> 현실 로또 생방송을 재현한 WebGL 기반 3D 시뮬레이터

## 🚀 Features
- Three.js + cannon-es 기반 3D 로또 머신
- 45개 공이 튀어오르고, 6개가 자동 정렬됨
- 번호별 오디오(TTS mp3) 재생
- UI minimal / 몰입형 애니메이션 연출
